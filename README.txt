# Sistema de Ventas de Seguros - Frontend (React)

## Requisitos
- Node.js 18+
- npm o yarn

## Instalación
1. Clona el repositorio:
   git clone https://github.com/tu-usuario/tu-repo.git

2. Ve a la carpeta del proyecto:
   cd tu-repo

3. Instala las dependencias:
   npm install

4. Crea un archivo `.env` y configura las variables necesarias:
  URL=http://localhost:3000

5. Ejecuta el proyecto:
   npm run start
6. Cambiar la ruta
   En servicios cambiar la ruta


# Sistema de Ventas de Seguros - Backend (.NET Core 8+)

## Requisitos
- .NET SDK 8+
- SQL Server 

## Instalación
1. Clona el repositorio:
   git clone https://github.com/tu-usuario/tu-repo.git

2. Ve a la carpeta del proyecto:
   cd tu-repo

3. Restaura las dependencias:
   dotnet restore

4. Configura la base de datos en `appsettings.json`.

5. Aplica las migraciones (si usas Entity Framework):
   dotnet ef database update

6. Ejecuta el proyecto:
   dotnet run

7. Crea un archivo `.env` y configura las variables necesarias:
  URL=https://localhost:7103 en 

8. Program.cs 
	Cambiar la el host para frontend
